## 3. Endpoints
````
JUEGOS

GET    -> /api/juegos
       -> /api/juegos/{nombre}
POST   -> /api/juegos
DELETE -> /api/juegos/{nombre}

MENSAJES

GET    -> /api/mensajes
PUT    -> /api/mensajes/{codigo}
DELETE -> /api/mensajes/{nombre}


SALAS

GET    -> /api/salas
       -> /api/salas/{NombreJuego}
POST   -> /api/salas
PUT    -> /api/salas/{codigo}
DELETE -> /api/salas/{codigo}

USUARIOS

GET    -> /api/usuarios/{id}
POST   -> /api/usuarios
PUT    -> /api/usuarios/{codigo}
DELETE -> /api/usuarios/{codigo}
````
